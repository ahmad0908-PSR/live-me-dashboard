# Live M&E Dashboard (Streamlit + Google Sheets)

This project builds a real-time M&E dashboard using:

- Python 3.13
- Streamlit
- Google Sheets as data source
- Auto-refresh & deployment on Streamlit Cloud

### Project Roadmap
Phase 0 — Project skeleton  
Phase 1 — Google Service Account setup  
Phase 2 — Google Sheets connection  
Phase 3 — Data cleaning  
Phase 4 — Dashboard UI  
Phase 5 — Auto-refresh  
Phase 6 — Deployment  
Phase 7 — Authentication  
Phase 8 — Advanced features
